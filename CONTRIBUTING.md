# Contributing

Contributions are welcome.

## How to contribute

1. Fork the repo
2. Create a branch: `git checkout -b my-feature`
3. Make your changes
4. Test against a real ROS2 workspace
5. Submit a pull request

## What to work on

Check the roadmap in the README — C++ support and AI-assisted 
dynamic topic resolution are the priorities right now.

## Reporting bugs

Open an issue with the raw output from ros2grapher and the 
ROS2 workspace structure that caused it.
